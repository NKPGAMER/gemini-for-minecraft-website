Gemini Assistant for Minecraft Bedrock Edition

Bạn muốn tích hợp 1 trợ lý vào Minecraft để hỗ trợ trong quá trình chơi? Hay cảm thấ cô đơn vì chơi một mình. Hãy thử dùng Gemini assistant, một trợ lý có thể cung cấp cho bạn những thông tin cần thiết và có thể trò chuyện với bạn.

Tải xuống & cấu hình

  • Tải Gemini Assistant

    • Nhấn vào đây để tải xuống.

  • Lấy API

    • Đi đến trang của Gemini API hoặc nhấn vào đây
    • Đăng nhập tài khoản
    • Nhấn vào 'Tạo khoá API' (Create API key) và làm theo các yêu cầu để lấy key

  • Cấu hình

    • Giải nén tệp vừa tải xuống
    • Mở thư mục vừa giải nén. Vào thư mục scripts và mở tệp config.js
    • Dán API vào cho config['API'].
      Mẫu: config['API'] = "ENTER_YOUR_API"
      => config['API'] = "abc123456789"
    • Có thể tuỳ chỉnh một vài thứ tại đó
    • Lưu tệp

Nhập Gemini vào Minecraft

  • Thiết lập Bedrock Server

    Dành cho chạy server local, bỏ qua nếu dùng cho các server online
    • Tải Bedrock Server từ trang chủ của Minecraft
    • Giải nén tệp vừa tải về
    • Khởi chạy bedrock_server.exe để thiết lập các dữ liệu cần thiết

  • Kích hoạt Module

    Chú ý: Nếu thiếu module thì Gemini sẽ không hoạt động
    • Mở thư mục của bedrock_server
    • Đi đến thư mục config/default/
    • Mở tệp permissions.json

    Bên trong tệp permissions.json

    {
  "allowed_modules": [
    "@minecraft/server-gametest",
    "@minecraft/server",
    "@minecraft/server-ui",
    "@minecraft/server-admin",
    "@minecraft/server-editor"
  ]
}

    Nếu không có @minecraft/server-net hãy thêm vào. Kết quả sẽ như sau:

    {
  "allowed_modules": [
    "@minecraft/server-gametest",
    "@minecraft/server",
    "@minecraft/server-ui",
    "@minecraft/server-admin",
    "@minecraft/server-editor",
    "@minecraft/server-net"
  ]
}
    Sau khi thêm nhớ kiểm tra lại cho đúng nha. Hoặc copy ở trên rồi paste vào cũng được :)

  • Thiết lập thế giới

    • Tạo một thế giới mới hoặc sao lưu thế giới hiện có
    • Trong phần tuỳ chỉnh thế giới hãy bật tất cả tính năng thử nghiệm
    • Sau khi đã có thế giới thì đến thư mục chứa thế giới các thế giới
    • Sao chép thế giới
    • Đến thư mục của Server và vào thư mục world
    • Tại đây nếu có thư mục Bedrock level thì xoá nó đi
    • Dán thế giới đã vào và đổi tên thành Bedrock level

  • Đưa Gemini Assistant vào server

    • Sao chép thư mục của Gemini Assistant mà ta vừa thiết lập bên trên
    • Đến thư mục của Server và dán vào development_behavior_packs
    • Quay lại và vào thư mục worlds/Bedrock level
    • Tại đây nếu chưa có tệp world_behavior_packs.json hãy tạo nó
    • Mở tệp world_behavior_packs.json

    Bên trong tệp world_behavior_packs.json:

    Nếu là tệp mới tạo hãy dán mã sau vào:
    [
  {
    "pack_id" : "eb75f2f2-aa22-4cba-8fc1-addcab8ac9da",
   "version" : [ 1, 0, 0 ]
  }
]
    Nếu đã có tệp và bên trong tệp đã có []. Chỉ cần thêm mã sau vào []:
    {
  "pack_id" : "eb75f2f2-aa22-4cba-8fc1-addcab8ac9da",
  "version" : [ 1, 0, 0 ]
}
  • Khởi động server
Nếu trong console có báo 
[SERVER] Pack Stack - [00][§l§bGemini §fArtificial intelligence] [packid_ver][eb75f2f2-aa22-4cba-8fc1-addcab8ac9da_1.0.0] [path][development_behavior_packs/Gemini AI]
Thì bạn đã làm đúng rồi đấy :)
